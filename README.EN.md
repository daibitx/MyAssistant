MyAssistant is a lightweight AI-powered chat assistant built with Blazor.
It provides a clean conversational interface, integrates with AI models, and supports building and querying a local knowledge base for enhanced contextual responses.

This project demonstrates how to:

Build an AI chat UI with Blazor

Connect to external or local AI models

Create and manage a custom local knowledge base

Combine AI responses with domain-specific content

The project is experimental and intended for learning, prototyping, and personal use.
